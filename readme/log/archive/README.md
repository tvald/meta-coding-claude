# Archive

Overflow from living documents, moved here when budgets are exceeded (see
[knowledge management](../../knowledge/management.md)), plus raw ingestion sources kept
after their content was filed, and retired specs/tasks (see
[work conventions](../../work/README.md)). Budget overflow is named
`<source-filename>-YYYY-MM.md`; other files keep their names. Archived content is
history — never edited, rarely read, kept because deletion destroys the ability to
answer "why did we think that?" Exception: queued *work* is never archived — it flows to
the backlog instead.
