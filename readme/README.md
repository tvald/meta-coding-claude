# readme/

<!-- PROCESS doc: two-line orientation for anyone landing here. -->

Two kinds of content live here:

- **[`meta/`](meta/README.md)** — the framework itself: process, standards, roles,
  templates, and the pristine `seed/` state tree. A project treats it as immutable;
  it changes only via [refinement](meta/process/refinement.md).
- **Everything else** (`knowledge/`, `standards/derived.md`, `work/`, `log/`) — this
  project's mutable state, created from `meta/seed/` on first run. Deleting it resets
  the framework for a new project.

Deploy, bootstrap, and reset procedures: [meta/README.md](meta/README.md#deploy-bootstrap-reset).
